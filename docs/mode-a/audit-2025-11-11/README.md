# Mode A Audit Package — 2025-11-11

- `report.md` – End-to-end audit narrative and remediation roadmap.
- `mode-a-guards.md` / `mode-a-guardrail-checklist.md` – Guardrail overview plus the evidence checklist captured during the review.
- `sbom-and-licenses.md` – Dependency bill of materials and license summary for touched surfaces.

- `open-questions.md` – Outstanding decisions or follow-up items.

- `pr-summary.md` – Changelog of the mitigation pull requests that landed during the audit.
- `findings.json` / `fixes.patch` – Machine-readable findings list and companion patch showing the implemented fixes.

Place any future Mode A audit deliverables in this directory to keep the repository index tidy.
